## P104

```html
<audio src="" controls loop autoplay muted>音频或视频</audio>
// controls：出现控制栏（注意这个不能缺否则无法显示）
// loop：自动循环
// autoplay：自动播放（有的浏览器会默认关闭写了也没用，需要用js改）
// muted：静音
```



## P105
